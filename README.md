# YOLOv10 论文复现

# 项目介绍

本项目基于 YOLOv10 目标检测模型，针对 VOC 数据集开展训练与测试工作，主要用于学习和研究目标检测算法的实践应用。项目包含完整的训练配置、数据集路径配置及测试效果验证相关代码，可直接基于本仓库代码复现 YOLOv10 在 VOC 数据集上的训练流程。

# 本项目主要内容

基于原始 YOLOv10 代码，本项目主要进行以下适配性修改（均为开源可见）：

1. 数据集配置优化：修改 `voc.yaml` 文件中的数据集路径，指定本地自定义数据集目录（`C:\Users\bimoh\Downloads\yolov10\datasets\VOC`），解决路径关联错误问题；

2. 训练参数调整：适配 RTX 4060 Laptop GPU 环境，设置合理的训练参数（`batch=8, epochs=10, device=0`），确保 GPU 正常加速训练；
3. 使用test_yolov10.py进行了一次预测，和使用命令yolo detect train model=yolov10s.pt data=voc.yaml epochs=10 batch=8 device=0运行十轮训练，查看 `runs/detect` 目录下的训练日志、性能指标图表及测试结果可视化文件，核心性能指标（mAP@0.5、精确率、召回率等）。
